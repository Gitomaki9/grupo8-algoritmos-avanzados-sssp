## 📋 Descripción del Proyecto

Este proyecto compara el algoritmo clásico de **Dijkstra** con el nuevo algoritmo **BMSSP** (Bounded Multi-Source Shortest Path) propuesto por **Duan, Mao, Shu y Yin (2025)**, que rompe la barrera del ordenamiento con complejidad:

$$
O(m \log^{2/3} n)
$$

El objetivo es evaluar experimentalmente ambos algoritmos en diferentes datasets para determinar en qué escenarios BMSSP supera a Dijkstra.
